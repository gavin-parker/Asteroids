# Asteroids
## Requirements
    * LibCinder https://libcinder.org/
    * OpenGL drivers
    * C++14 compatible compiler (GCC/Clang)
    * Cmake 3.13+
## Compiling & running

To compile:
    `mkdir build && cd build`
    `cmake .. -DCINDER_PATH=~/Cinder`
    `make`
To run:
    `./asteroids`

## Features
    * Moving ship (with lasers!)
    * Randomly spawning asteroids
